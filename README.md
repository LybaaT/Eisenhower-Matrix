# Eisenhower Matrix Tool

A decision-support tool based on the Eisenhower Matrix framework, designed to help users prioritize tasks by urgency and importance.

## Overview

The Eisenhower Matrix is a time management framework that categorizes tasks into four quadrants:

| Quadrant | Description | Recommended Action |
|----------|------------|-------------------|
| Urgent and Important | Time-sensitive, high-impact tasks | Execute immediately |
| Not Urgent and Important | Long-term, strategic tasks | Schedule |
| Urgent and Not Important | Tasks requiring immediate attention but low impact | Delegate |
| Not Urgent and Not Important | Low-value activities | Eliminate |

This tool provides a structured interface to apply this framework in practice, enabling more deliberate and effective prioritization.

## Features

- Four-quadrant task organization  
- Simple task input and classification  
- Clear prioritization logic based on urgency and importance  
- Lightweight and responsive interface
